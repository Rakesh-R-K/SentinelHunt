# Contribution Guidelines for SentinelHunt

Thank you for considering contributing to SentinelHunt!

## How to Contribute
- Fork the repository and create your branch from `main`.
- Add your code or documentation changes.
- Ensure your code passes all tests and follows the project style.
- Submit a pull request with a clear description of your changes.

## Areas to Contribute
- Detection rules and intelligence modules
- Machine learning models and evaluation
- Integration and unit tests
- Documentation and tutorials

## Code of Conduct
Please be respectful and constructive in all interactions.
